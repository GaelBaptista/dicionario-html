Tags de Inicialização

<!DOCTYPE html>

A tag <!DOCTYPE html> define o tipo de documento e a versão do HTML.

<!DOCTYPE html>

<html>

A tag <html> é a raiz do documento HTML. Todo o conteúdo da página deve estar dentro dessa tag.

<html lang="pt-BR">

<head>

A tag <head> contém metadados e links para arquivos externos, como CSS e JavaScript.

<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
</head>

<template>

A tag <template> define um fragmento de HTML que pode ser reutilizado e instanciado dinamicamente usando JavaScript.

<template id="my-template">
  <div>
    <p>Este é um template.</p>
  </div>
</template>

<script>
  const template = document.getElementById('my-template');
  const clone = document.importNode(template.content, true);
  document.body.appendChild(clone);
</script>

<meta>

A tag <meta> fornece metadados sobre o documento, como a codificação de caracteres.

<meta charset="UTF-8">

Tags de Estrutura

<header>

A tag <header> define uma área de cabeçalho na página, geralmente contendo logotipos e navegação.

<header>...</header>

<nav>

A tag <nav> define uma área de navegação com links para outras páginas ou seções do site.

<nav>...</nav>

<main>

A tag <main> é usada para encapsular o conteúdo principal da página, excluindo cabeçalhos e rodapés.

<main>...</main>

<footer>

A tag <footer> define uma área de rodapé na página, geralmente contendo informações de contato e direitos autorais.

<footer>...</footer>

<section>

A tag <section> é usada para agrupar conteúdo relacionado em uma seção temática.

<section>...</section>

<article>

A tag <article> define um conteúdo independente que pode ser distribuído ou reutilizado em outros contextos.

<article>...</article>

<aside>

A tag <aside> define conteúdo complementar, como barras laterais ou informações adicionais.

<aside>...</aside>

<slot>

A tag <slot> é usada em Web Components para definir um espaço onde o conteúdo pode ser inserido.

<my-component>
  <span slot="header">Cabeçalho</span>
  <span>Corpo do conteúdo</span>
</my-component>

<template id="my-component-template">
  <div>
    <slot name="header"></slot>
    <slot></slot>
  </div>
</template>

Tags de Formulários

<form>

A tag <form> cria um formulário para entrada de dados do usuário.

<form>
    <input type="text" name="nome">
    <input type="submit" value="Enviar">
</form>

<input>

A tag <input> cria campos de entrada em um formulário, como caixas de texto e botões de envio.

<input type="text" name="nome">
<input type="submit" value="Enviar">

<label>

A tag <label> define um rótulo para um campo de entrada. O atributo <for> vincula o rótulo ao campo.

<label for="nome">Nome:</label>
<input type="text" id="nome" name="nome">

Tags de Multimídia

<img>

A tag <img> exibe uma imagem na página. O atributo <src> define o caminho da imagem e <alt> fornece uma descrição alternativa.

<img src="imagem.jpg" alt="Descrição da imagem">

<audio>

A tag <audio> incorpora áudio na página. O atributo <controls> adiciona controles de reprodução.

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
</audio>

<picture>

A tag <picture> fornece uma maneira de exibir diferentes imagens com base em condições, como tamanho da tela.

<picture>
    <source srcset="imagem-small.jpg" media="(max-width: 600px)">
    <source srcset="imagem-large.jpg" media="(min-width: 601px)">
    <img src="imagem-default.jpg" alt="Descrição da imagem">
</picture>

Tags de Tabelas

<table>

A tag <table> cria uma tabela. Ela deve conter pelo menos uma linha (<tr>) e pode incluir cabeçalhos (<th>) e células de dados (<td>).

<table>
    <tr>
        <th>Cabeçalho 1</th>
        <th>Cabeçalho 2</th>
    </tr>
    <tr>
        <td>Dado 1</td>
        <td>Dado 2</td>
    </tr>
</table>

<object>

A tag <object> incorpora outros tipos de dados, como imagens ou vídeos, em um documento HTML.

<object data="documento.pdf" 
type="application/pdf" 
 width="600" height="400">
    <p>Seu navegador não suporta a visualização de PDFs. 
Baixe o documento <a href="documento.pdf">aqui</a>.</p>
</object>

<details> e <summary>

A tag <details> cria um controle que o usuário pode abrir ou fechar, e a tag <summary> define o título visível que pode ser clicado.

<details>
    <summary>Mais informações</summary>
    <p>Aqui estão mais detalhes.</p>
</details>

Tags Meta

<meta>

As tags <meta> fornecem metadados sobre o documento. O atributo <charset> define a codificação de caracteres, enquanto <name> pode fornecer outras informações.

<meta charset="UTF-8">
<meta name="description" content="Descrição do site">
